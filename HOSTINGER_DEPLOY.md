# Deploy en Hostinger

Esta app debe arrancar desde la raiz del repositorio.

## Comando de inicio

```bash
npm start
```

## Build

Si Hostinger ejecuta fase de build, puede usar:

```bash
npm run build
```

No compila assets; solo valida el flujo de despliegue.

## Comprobacion rapida

- `/` debe cargar la app
- `/api/health` debe devolver JSON
